# organizador-pessoal
Organizador Pessoal

Como usar:
1. Extraia o ZIP.
2. Abra o arquivo index.html no navegador.
3. Os dados ficam salvos no navegador usando localStorage.

Recursos:
- Categorias em select.
- Cadastro de categorias em popup/modal.
- Botão com ícone para adicionar categoria.
- Dashboard separado para Tarefas.
- Dashboard separado para Compras.
- Lista de compras com total de itens e itens comprados.
- Backup e importação JSON.
